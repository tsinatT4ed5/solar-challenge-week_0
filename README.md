# Solar Challenge Week 0 - KAIM 8

This repository contains my Week 0 submission for the 10 Academy KAIM 8 program.
The project focuses on exploratory data analysis of solar radiation datasets from
Benin, Togo, and Sierra Leone to identify high-potential regions for solar installation.

### Contents
- `notebooks/solar_analysis.ipynb` — main EDA and comparison notebook
- `reports/week0_report.pdf` — short summary report
- `dashboard_screenshots/` — screenshots of Streamlit dashboard

### Tools Used
Python, Pandas, Matplotlib, Seaborn, Streamlit
